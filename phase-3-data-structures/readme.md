## 🎯 Objective

Develop strong **data manipulation and algorithmic thinking** through:

- Array traversal and transformations  
- Searching and optimization strategies  
- Efficient data rearrangement  
- Frequency analysis  
- String manipulation without shortcuts  

This phase introduces the **first true data structures** and trains the mind to think in **time complexity, memory layout, and structured data operations.**

## 📂 Data Structure Drills

### 🔹 Arrays

- [x] Find second largest element  
- [ ] Remove duplicates from array  
- [ ] Rotate array left/right  
- [ ] Find missing number in 1–N  
- [ ] Move all zeros to end  
- [ ] Kadane’s Algorithm (Maximum subarray sum)  
- [ ] Merge two sorted arrays  
- [ ] Implement Binary Search  
- [ ] Count frequency of elements  
- [ ] Stock buy-sell (maximum profit)

### 🔹 Strings

- [ ] Check anagram  
- [ ] Reverse string without built-in reverse  
- [ ] Longest word in a sentence  
- [ ] Count vowels and consonants  
- [ ] Remove spaces  
- [ ] Check substring manually  
- [ ] String compression (aaabb → a3b2)  
- [ ] Longest palindrome substring (brute force)
